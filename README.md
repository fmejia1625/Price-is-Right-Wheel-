# Price Is Right Wheel

A small browser-based demo that simulates the "Price Is Right" prize wheel.

Features
- Interactive wheel controlled in `wheel.js`
- Simple styling in `styles.css`
- Optional sound effects in the `audio/` folder

How to run
- Open `index.html` in a web browser (double-click or serve from a local webserver).

Development
- Edit the wheel logic in `wheel.js`.
- Update styles in `styles.css`.
- Add or replace sound files in the `audio/` folder. A helper script `audio/fetch_sfx.ps1` is included for sourcing effects.

Files
- `index.html` — main page
- `wheel.js` — wheel behavior and controls
- `styles.css` — styles
- `audio/` — sound assets and scripts
