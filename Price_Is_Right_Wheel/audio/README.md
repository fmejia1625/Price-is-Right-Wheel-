Place show-style SFX files into this folder using the filenames expected by the game (lowercase):

- spin.mp3      — spin start/impact
- bust.mp3      — buzzer or negative sting
- mega.mp3      — $1.00 fanfare
- applause.mp3  — crowd applause/cheer
- tieStart.mp3  — short tension sting when tie-break begins
- tieWin.mp3    — short celebratory sting for tie-break winner
- bonus.mp3     — bonus spin cue
- click.mp3     — UI button click
- tick.mp3      — wheel peg tick (already referenced by index.html)

Suggested free SFX sources:
- Freesound.org (Creative Commons) — check licenses
- Pixabay / FreeSoundEffects / Zapsplat — many royalty-free options

Drop files here using the exact names above. If a file is missing the game will fall back to lightweight WebAudio beep stings.
